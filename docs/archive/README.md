# Archive

Este diretorio guarda documentacao que nao e mais usada como referencia principal do projeto, mas que ainda pode ser util como historico.

## Criterio de arquivamento

Foram movidos para ca:

- checklists antigos de fases e runtimes;
- TODOs superados por planos mais novos;
- artefatos auxiliares de PR.
- documentacao de runtimes legadas substituidas por superficies oficiais mais novas;
- analises estrategicas e leituras criticas que hoje servem mais como historico;
- explicacoes tecnicas de versoes antigas que ja nao sao a referencia principal.

## Estrutura

- `checklists/`: checklists antigos de implementacao.
- `planning/`: TODOs e planejamentos superados.
- `pr/`: artefatos auxiliares de PR.
- `legacy_runtimes/`: documentacao de runtimes substituidas.
- `analysis/`: analises historicas.
- `technical_history/`: explicacoes de versoes antigas.

## O que continua fora do arquivo

Permanece na raiz de `docs/` o que ainda serve como:

- plano principal;
- status tecnico atual;
- contrato operacional;
- referencia funcional ou historica ainda consultavel.
