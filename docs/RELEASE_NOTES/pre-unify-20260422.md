## Impacto e compatibilidade
- Endpoints read-only do `runtime v2` (agregação, conversão, estoque, fisconforme) não apresentam quebras intencionais.
- Mudanças visam estabilidade de CI/build; desenvolvedores que dependem de `pip install -e .` localmente devem adotar o workaround ou aplicar a correção permanente.
